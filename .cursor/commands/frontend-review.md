Review all code relating to the frontend (UI):

- Review frontend code for WCAG 2.2 AA compliance (keyboard navigation, focus management, ARIA roles, color contrast).
- Review frontend code for suitability for desktop and tablet screen sizes (including iPad Mini and 4K screens).
- Is the UI/UX intuitive for a user?
- Has the color scheme and styling been used consistently throughout?
- If there is a dark-mode option has the color scheme and styling been used consistently throughout in this mode?
- If there is a dark-mode option, is it WCAG 2.2 AA compliant? Pay attention to color contrast.
